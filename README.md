## Currency converter
The first currency calculator I made allows you to convert the amount in PLN into one of six currencies:
 1. USD
 2. EURO
 3. CHF
 4. GBP
 5. CZK
 6. RMB

## Features

 - HTML
 - CSS
 - JavaScript
 - BEM convention
 - Normalize
